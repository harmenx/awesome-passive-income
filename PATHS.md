# Passive Income Starter Paths

This document outlines high-leverage passive income paths, recommended tools, and suggested first steps. Choose one path and follow the roadmap to get started.

---

## 1. Faceless YouTube / Shorts

**Goal:** Generate revenue via ads and affiliate links with minimal personal exposure.  
**Effort Level:** Beginner → Intermediate  
**Automation Potential:** High  

### Tools
- **CapCut** – Video editing and templating  
- **Pictory / Runway** – Auto-generate videos from scripts  
- **Hootsuite / Buffer** – Schedule posting  
- **TubeBuddy / VidIQ** – SEO optimization  

### First Steps
1. Pick a niche (tech, finance, productivity, AI tools)  
2. Write 5–10 short scripts (~30–60s)  
3. Generate videos using Pictory / CapCut templates  
4. Schedule publishing via Hootsuite  
5. Track CTR and watch time  
6. Iterate based on metrics  

---

## 2. Hyper-Casual Mobile Games

**Goal:** Monetize via ads in simple, addictive mobile games.  
**Effort Level:** Intermediate  
**Automation Potential:** Medium  

### Tools
- **Unity / Godot** – Game engine  
- **AppLovin / AdMob** – Monetization  
- **Firebase / PlayFab** – Analytics & retention  
- **Figma / Canva** – Quick asset creation  

### First Steps
1. Build 1–2 prototype games with simple mechanics  
2. Integrate AdMob or AppLovin ads  
3. Test with friends / small audience  
4. Launch on iOS/Android  
5. Track D1/D7 retention and ARPDAU  
6. Iterate and replicate  

---

## 3. Micro-SaaS / Automation Tools

**Goal:** Sell a small software solution solving a niche pain point.  
**Effort Level:** Intermediate → Advanced  
**Automation Potential:** High  

### Tools
- **Vercel / Railway / Render** – Deployment  
- **Stripe / Paddle** – Payments  
- **n8n / Zapier** – Workflow automation  
- **Notion / Airtable / Google Sheets** – Backend or MVP DB  

### First Steps
1. Identify a small, recurring pain point for a niche audience  
2. Build a minimal MVP  
3. Offer a free trial or beta to validate demand  
4. Automate onboarding, billing, and delivery  
5. Gather feedback and iterate  

---

## 4. Affiliate Marketing / Content Monetization

**Goal:** Earn commissions by promoting products via blogs, newsletters, or social media.  
**Effort Level:** Beginner → Intermediate  
**Automation Potential:** Medium  

### Tools
- **WordPress / Ghost / Medium** – Blog or newsletter  
- **Amazon Associates / Impact / ShareASale** – Affiliate programs  
- **Mailchimp / ConvertKit** – Email campaigns  
- **Canva / Figma** – Content creation  

### First Steps
1. Choose a niche with high affiliate potential  
2. Build a small blog or newsletter  
3. Write 5–10 high-value posts with affiliate links  
4. Promote via social media or organic search  
5. Automate newsletters and track clicks/conversions  

---

## 5. Crypto / DeFi / NFT

**Goal:** Generate passive income via staking, farming, or trading automation.  
**Effort Level:** Intermediate → Advanced  
**Automation Potential:** Medium → High  

### Tools
- **MetaMask / Ledger / Trezor** – Wallets  
- **DeFi Platforms (Aave, Compound, Uniswap)** – Staking / yield  
- **CoinGecko / DEXTools** – Analytics  
- **Python / Node.js scripts** – Automation  

### First Steps
1. Choose low-risk staking/farming opportunities  
2. Allocate a small test amount  
3. Set up alerts and monitor returns  
4. Automate simple trading or farming strategies  
5. Track ROI and scale gradually  

---

> **Tip:** Pick one path and fully focus for 2–4 weeks before adding another. Measure leading indicators (CTR, ARPDAU, MRR, ROI) and iterate weekly.

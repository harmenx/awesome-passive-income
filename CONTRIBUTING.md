# Contributing to Awesome Passive Income

Thank you for your interest in contributing! This guide will help you add new tools, tutorials, resources, or case studies to the repo in a consistent and useful way.

---

## 1. How to Contribute

You can contribute in the following ways:

- **Add a new tool** to `TOOLS.md`
- **Add a new tutorial or case study** to `RESOURCES.md`
- **Add a new path or roadmap** to `PATHS.md`
- **Fix typos or formatting** in any Markdown file
- **Suggest improvements** via GitHub Issues

---

## 2. Guidelines for Adding Tools

When adding a tool to `TOOLS.md`, include the following information:

| Field | Example |
|-------|---------|
| Tool | CapCut |
| Use Case | Video editing & templating |
| Path | Faceless YouTube / Shorts |
| Notes | Free, easy workflow, supports automation |

- Place the tool in the correct category (Video, Automation, Games, etc.)
- Keep descriptions concise (1–2 lines)
- Add links if available, but optional

---

## 3. Guidelines for Adding Tutorials / Case Studies

When adding a tutorial or case study to `RESOURCES.md`, include:

- **Title of tutorial/case study**
- **URL link**
- **Path** (which passive income stream it applies to)
- **Short description** (1–2 lines)

- **[Pictory AI Tutorial](https://www.youtube.com/watch?v=9vWv1j7mX-8)** – Auto-generate YouTube Shorts from scripts. Path: Faceless YouTube / Shorts
